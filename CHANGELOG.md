# Changelog

## [1.0.0] - Unreleased

### Added
- Added the `/baublecatalog` client command.
- Added timestamped CSV exports under `config/baublecatalog/`.
- Added Bubbles-recognized Bauble variant discovery.
- Added required CSV columns for registry ID, metadata, display name, source mod ID, and Bauble type ID.
- Added metadata columns for source mod name, Bauble type name, rarity, tooltip, creative tab, max stack size, max durability, and ore dictionary names.
- Added chat feedback with exported row count and output path.

### Compatibility
- Supports Minecraft 1.12.2 modpacks using Bubbles.
- Original Baubles and BaublesEX are not supported targets.
